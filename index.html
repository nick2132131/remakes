Getting started

Setup in 4 steps.

01
Download the .exe

Click the download button above. Save the file anywhere — Desktop works fine. Windows may show a SmartScreen warning because the binary isn't signed yet. Click More info → Run anyway.

02
Run as Administrator

The app requests admin automatically via its UAC manifest. Accept the prompt. QuorumAPI needs elevated privileges to inject into Roblox.

03
Open Roblox first

Launch Roblox and join any game. Wait until you're fully in-game before attaching. The executor detects RobloxPlayerBeta.exe automatically.

04
Attach and execute

Click Attach. Wait for the "Attached" status. Paste or write your script in the editor, then hit Execute. That's it.
